# Weather Web App 🌦️

A simple weather web application built with Node.js and React.js.

## Features 🚀

- Real-time weather updates
- User-friendly interface
- Responsive design

## Technologies Used 💻

- **Backend**: Node.js, Express.js
- **Frontend**: React.js
- **API**: OpenWeatherMap, Data Gov, Nominatim
- **Styling**: Tailwind CSS
- **HTTP Requests**: Axios
